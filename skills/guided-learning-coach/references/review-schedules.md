# Review Schedules

Generate review tasks based on what the learner can already explain and where they still hesitate.

## Default Cadence

Use this cadence unless the learner asks for something lighter:

- `1 天后`: Re-state the core idea from memory and give one example.
- `3 天后`: Compare it with a nearby concept or common misconception.
- `7 天后`: Apply it to a fresh scenario or judgment problem.
- `14 天后`: Summarize the idea, explain why it matters, and connect it to the larger route.

## Adapt by Mastery

### 基础掌握

Use when the learner can explain the idea and give a reasonable example, but still hesitates on comparison or transfer.

- Keep all four checkpoints.
- Make tasks shorter and more concrete.
- Revisit the same core distinction at least twice.

### 深层掌握

Use when the learner can compare, critique, or transfer confidently.

- Keep the 3-day and 7-day reviews.
- Make the 14-day review synthetic and cross-topic.
- Use broader questions instead of definition checks.

## Adapt by Failure Pattern

- If the learner confuses neighboring concepts, prioritize comparison tasks.
- If the learner can recite but not apply, prioritize transfer tasks.
- If the learner freezes under open questions, give smaller prompts first and widen later.
- If the learner raises a strong objection, include one review task that revisits the disagreement explicitly.

## Review Task Style

- Prefer short, answerable prompts over long reading assignments.
- Keep review tied to the original learning goal.
- If the learner asked for no evaluation, still provide review tasks without a score.
