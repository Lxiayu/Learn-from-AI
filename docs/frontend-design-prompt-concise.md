# AI辅助学习平台前端设计提示词（精简版）

## 项目背景
基于"Guided Learning Coach"技能，打造简约但功能强大的AI学习应用，采用苏格拉底式提问、间隔复习策略，帮助用户系统掌握概念性知识。

## 核心功能模块
1. **智能学习对话系统**：苏格拉底式对话、问题阶梯（解释→举例→比较→迁移）、实时纠错
2. **个性化学习路线**：路线图生成、任务卡、进度可视化
3. **间隔复习系统**：1/3/7/14天复习节奏、掌握度追踪
4. **学习数据看板**：时长统计、成就系统

## 设计原则
- **极简主义**：去除冗余，清晰视觉层级，充足留白
- **对话优先**：聊天界面为核心，单任务聚焦
- **智能体验**：AI驱动个性化，上下文感知
- **无障碍**：深色模式、可调节字号、高对比度

## UI设计规范
**色彩**：主色#3A86FF（智慧蓝），辅助色#FF6B35（活力橙），中性色#2D3436/#F8F9FA
**字体**：SF Pro / PingFang SC，16-18px正文
**间距**：8px基础单位，模块间距≥24px
**圆角**：4-16px，轻量阴影

## 核心页面设计
1. **首页**：个性化问候、当前任务卡、复习提醒
2. **对话界面**：清晰的AI/用户气泡、简洁输入框、提示/跳过选项
3. **学习路线**：垂直时间线，已完成/进行中/未解锁状态区分
4. **复习计划**：今日待复习优先、进度可视化、日历视图
5. **个人中心**：成就展示、学习统计、简洁设置

## 交互设计
- **动效**：0.3秒页面切换，按钮缩放，进度条渐进填充
- **手势**：左滑返回，下拉刷新，Enter发送，Esc关闭
- **反馈**：即时状态变化，AI思考加载动画，Toast提示

## 响应式设计
- 移动端：<768px，底部导航，单列布局
- 平板：768-1024px，侧边导航，分屏展示
- 桌面：>1024px，固定侧边栏，三列布局，快捷键

## 技术建议
- 框架：Flutter（跨平台）
- 状态管理：Provider/Riverpod
- 组件：Material Design 3 + 自定义
- 优化：预加载、懒加载、虚拟滚动、离线缓存

## 参考案例
- Socratic by Google：多种输入、视觉化解释
- ChatGPT：流畅对话体验
- Duolingo：游戏化激励
- Anki：科学间隔复习

## 设计核心理念
"简约但功能强大"——通过极简界面、流畅对话、智能体验，让用户专注学习本身。关键决策：对话式UI为核心、渐进式信息展示、智能化个性化、游戏化激励、科学复习系统。
