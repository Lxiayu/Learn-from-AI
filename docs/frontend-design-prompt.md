# AI辅助学习平台前端设计提示词

## 项目背景

我们要开发一款基于"Guided Learning Coach"技能的AI辅助学习平台。该平台采用苏格拉底式提问、间隔复习和主动学习策略，帮助用户系统掌握概念性知识。设计理念是"简约但功能强大"，打造一个专注、高效、智能的学习体验。

## 核心功能模块

### 1. 智能学习对话系统
- **苏格拉底式对话界面**：AI教练通过引导性问题帮助用户主动思考
- **问题阶梯展示**：解释 → 举例 → 比较 → 迁移的渐进式学习路径
- **实时反馈与纠错**：智能识别误解并提供针对性讲解

### 2. 个性化学习路线
- **学习路线图生成**：根据用户目标和水平生成个性化学习计划
- **当前任务卡**：清晰展示当前学习目标和通过标准
- **进度可视化**：直观展示学习进度和掌握程度

### 3. 间隔复习系统
- **科学复习计划**：基于遗忘曲线的1天、3天、7天、14天复习节奏
- **复习提醒**：智能推送复习任务
- **掌握度追踪**：基础掌握 vs 深层掌握的可视化区分

### 4. 学习数据看板
- **学习时长统计**：可视化学习时间分布
- **知识点掌握度**：各知识点的掌握程度展示
- **学习成就系统**：勋章、连续学习天数等激励机制

## 设计原则

### 1. 极简主义设计
- **少即是多**：去除不必要的装饰元素，聚焦核心学习内容
- **清晰的视觉层级**：通过字号、颜色、间距建立明确的信息层级
- **留白艺术**：充足的留白空间，降低视觉负担，提升专注度
- **一致性**：统一的设计语言，相同功能使用相同的视觉表达

### 2. 对话式交互优先
- **聊天界面为核心**：类似ChatGPT的对话式UI，但针对学习场景优化
- **单任务聚焦**：一次只展示一个核心问题或任务
- **渐进式信息展示**：避免信息过载，按需展示详细内容
- **自然的对话流**：模拟真实师生对话的节奏和语气

### 3. 智能化体验
- **AI驱动的个性化**：根据用户行为动态调整界面和内容
- **上下文感知**：记住用户的学习状态和偏好
- **智能推荐**：主动推荐相关学习资源和复习任务
- **自适应布局**：根据内容类型自动调整最佳展示方式

### 4. 无障碍与包容性
- **深色模式支持**：保护视力，适应不同使用场景
- **可调节字号**：满足不同用户的阅读需求
- **高对比度选项**：确保视觉障碍用户也能正常使用
- **键盘导航支持**：支持快捷键操作，提升效率

## UI设计规范

### 色彩系统

```
主色调：智慧蓝 #3A86FF（传递专业、可信赖感）
辅助色：活力橙 #FF6B35（激励学习、突出行动）
中性色：深灰 #2D3436、浅灰 #F8F9FA
功能色：成功绿 #00C851、警告黄 #FFBB33、错误红 #FF3B30
```

**使用原则**：
- 主色调用于重要按钮、进度条、选中状态
- 辅助色用于强调、激励元素、通知徽章
- 中性色用于背景、文字、边框
- 功能色仅用于对应的功能场景

### 字体系统

```
标题字体：SF Pro Display / PingFang SC（粗体）
正文字体：SF Pro Text / PingFang SC（常规）
代码字体：SF Mono / JetBrains Mono

字号规范：
- 大标题：28-32px
- 小标题：20-24px
- 正文：16-18px
- 辅助文字：14px
- 小字：12px
```

### 间距系统

```
基础单位：8px
常用间距：8px, 16px, 24px, 32px, 48px
模块间距：≥24px
行间距：字号的1.5-2倍
卡片内边距：16-24px
```

### 圆角与阴影

```
圆角：4px（小组件）、8px（卡片）、12px（大卡片）、16px（模态框）
阴影：轻量阴影用于悬浮效果，避免过度使用
```

## 交互设计规范

### 1. 动效设计

**页面切换**：
- 时长：0.3秒
- 效果：平滑淡入淡出
- 避免过度炫技影响加载速度

**元素动效**：
- 按钮点击：轻微缩放（0.95倍）+ 颜色变化
- 卡片展开：平滑高度变化
- 进度条更新：渐进式填充动画
- 徽章解锁：粒子扩散效果

**反馈动效**：
- 正确答案：绿色闪烁 + 勾选动画
- 错误答案：轻微抖动 + 提示出现
- 任务完成：庆祝动画（可选）

### 2. 手势操作

**移动端**：
- 左滑返回上一页
- 下拉刷新
- 长按卡片显示快捷操作
- 双击点赞/收藏

**桌面端**：
- 支持键盘快捷键
  - Enter：发送消息
  - Esc：关闭模态框
  - Tab：切换焦点
  - Cmd/Ctrl + K：快速搜索

### 3. 反馈机制

**即时反馈**：
- 按钮点击：视觉状态变化
- 输入验证：实时提示
- 操作成功/失败：Toast提示

**延迟反馈**：
- AI思考中：加载动画 + 文字提示
- 数据加载：骨架屏展示
- 网络错误：友好的错误页面 + 重试选项

### 用户体验指标

- **易用性**：新用户能在3分钟内完成首次学习

## 总结

这款AI辅助学习平台的设计核心是"简约但功能强大"。通过极简的界面设计、流畅的对话式交互、智能化的学习体验，打造一个让用户专注于学习本身的平台。设计需要平衡功能丰富性与操作简洁性，确保用户能够轻松上手，同时享受深度的学习体验。

关键设计决策：
1. **对话式UI为核心**：模拟真实师生对话，降低学习门槛
2. **渐进式信息展示**：避免信息过载，保持界面清爽
3. **智能化个性化**：AI驱动的自适应学习体验
4. **游戏化激励**：成就系统增强学习动力
5. **科学的复习系统**：基于遗忘曲线的间隔复习

通过这些设计原则和规范，我们期望打造一款真正帮助用户高效学习的AI教育平台。
